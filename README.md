# umukoro
a todo app 
## Getting Started

First, clone repo

```bash
git clone <repo url>
# install dependencies
yarn 
```
```bash
# create env file and add your env variables
yarn 
```
run the json  server:

```bash
yarn json
# run nextjs app
yarn dev
```

Open [http://localhost:3000](http://localhost:3001) with your browser to see the result.
